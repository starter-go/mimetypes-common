# 扩展：

在原 CSV 文件3列的基础上：

    Name,Template,Reference

扩展为：

    Name,Template,Reference, ... ,Suffixes

例如：

    png,image/png,[W3C][PNG_Working_Group],(.png)

    html,text/html,[W3C][Robin_Berjon],(.html .htm)

每个后缀之间以 '.' 为分隔符
